# practice_for_markdown

# 1. 글자크기정하기

* 입력
```
# 제목1  
## 제목2  
### 제목3  
#### 제목4  
##### 제목5  
###### 제목6  
```
* 출력
-----
# 제목1  
## 제목2  
### 제목3  
#### 제목4  
##### 제목5  
###### 제목6 
-----

# 2. 목록(들여쓰기)

* 입력
```
1. 제목1  
2. 제목2  
    1. 제목(1)  
    2. 제목(2)  
3. 제목3
    1. 제목(1)
    2. 제목(2)  
        1. 제목((1))  
        2. 제목((2))    
```
※ 들여쓰기는 스페이스 4번
    

* 출력
---
1. 제목1  
1. 제목2  
    1. 제목(1)  
    2. 제목(2)  
1. 제목3  
    1. 제목(1)  
    1. 제목(2)  
        1. 제목((1))  
        1. 제목((2))  
---
